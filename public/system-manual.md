## 1. 系统概述

### 1.1 系统简介

非凡项目管理系统是一款全功能的项目管理平台，集成了项目管理、任务管理、客户管理、合同管理、订单管理、产品管理等核心功能，同时支持移动端APP访问、数据统计看板、Windows一键部署等高级功能。

### 1.2 技术架构

- **前端框架**: Next.js 16 + React 19
- **UI组件**: shadcn/ui + Tailwind CSS
- **后端框架**: Next.js API Routes
- **数据库**: PostgreSQL + Drizzle ORM
- **移动端**: React Native
- **桌面端**: Electron 40

### 1.3 系统特色

- 响应式设计，支持多端访问
- 实时数据同步
- 完善的权限管理
- 数据可视化看板
- 支持远程数据库配置
- 移动端APP支持

## 3. 功能模块介绍

### 3.1 项目看板

**功能说明**: 项目看板是系统的首页，展示项目统计信息和状态概览。

**主要功能**:
- 项目数量统计
- 任务数量统计
- 合同金额汇总
- 订单金额汇总
- 项目状态分布图
- 最近项目列表

**操作步骤**:
1. 登录系统后自动进入项目看板
2. 查看各项统计数据
3. 点击项目名称可快速跳转到项目详情

### 3.2 项目列表

**功能说明**: 管理所有项目信息，支持创建、编辑、删除项目。

**主要功能**:
- 项目列表展示
- 项目搜索和筛选
- 创建新项目
- 编辑项目信息
- 删除项目
- 导入导出项目数据

**操作步骤**:

**创建项目**:
1. 点击"新建项目"按钮
2. 填写项目基本信息（项目名称、描述、时间等）
3. 填写产品信息（物料编码、产品名称、规格等）
4. 填写客户信息
5. 填写团队成员信息
6. 填写合同和订单信息
7. 点击"保存"完成创建

**编辑项目**:
1. 在项目列表中找到目标项目
2. 点击"编辑"按钮
3. 修改需要更新的信息
4. 点击"保存"完成编辑

**删除项目**:
1. 在项目列表中找到目标项目
2. 点击"删除"按钮
3. 确认删除操作

### 3.3 项目详情

**功能说明**: 查看项目详细信息和管理项目下的任务。

**主要功能**:
- 项目基本信息展示
- 项目成员管理
- 任务列表
- 甘特图视图
- 项目附件管理

**操作步骤**:
1. 在项目列表中点击项目名称进入详情
2. 查看项目基本信息
3. 切换到"任务"标签管理任务
4. 切换到"甘特图"查看项目进度

### 3.4 任务管理

**功能说明**: 创建、编辑、删除任务，设置优先级和状态。

**主要功能**:
- 任务列表展示

- 创建新任务
- 编辑任务
- 删除任务
- 任务状态更新
- 任务分配

- 待办 (todo): 任务尚未开始
- 进行中 (in_progress): 任务正在进行
- 已完成 (completed): 任务已完成

**任务优先级**:

**操作步骤**:

**创建任务**:
1. 在项目详情页点击"新建任务"
2. 填写任务名称和描述
3. 设置任务优先级
4. 分配任务负责人
5. 设置计划开始和结束时间
6. 点击"保存"完成创建

1. 找到目标任务
2. 点击状态下拉菜单
3. 选择新的状态

3. 在目标电脑上点击"导入配置"

## 8. 数据库配置

### 8.1 功能说明

数据库配置功能允许用户配置远程数据库连接，实现多台电脑之间的数据同步。

### 8.2 使用场景

**场景一：远程共享数据库（自动同步）**
- 配置远程数据库连接
- 所有电脑连接同一远程数据库
- 数据实时自动同步

**场景二：设置导出/导入（手动同步）**
- 在一台电脑上配置并导出
- 在其他电脑上导入配置
- 手动同步配置信息

### 8.3 操作步骤

**配置远程数据库**:
1. 以管理员身份登录系统
2. 进入后台管理 > 数据库配置
3. 点击"新建配置"
4. 填写配置信息：
   - 配置名称：自定义名称
   - 主机地址：数据库服务器IP或域名
   - 端口号：数据库端口（默认5432）
   - 数据库名：数据库名称
   - 用户名：数据库用户名
   - 密码：数据库密码
5. 点击"测试连接"验证
6. 点击"保存"

**应用配置**:
1. 在配置列表中找到目标配置
2. 点击"应用"按钮
3. 确认切换
4. 系统将重启并连接新数据库

**导出/导入配置**:
1. 点击"下载配置"导出配置文件
2. 在目标电脑上点击"导入配置"
3. 选择配置文件
4. 应用导入的配置

### 8.4 注意事项

- 切换数据库后需要刷新页面
- 远程数据库需要网络连接稳定
- 请确保数据库安全性，使用强密码
- 建议定期备份数据库

## 9. 常见问题

### 9.1 登录问题

**Q: 忘记密码怎么办？**
A: 请联系系统管理员重置密码。

**Q: 登录后页面空白？**
A: 请清除浏览器缓存后重试，或检查网络连接。

### 9.2 权限问题

**Q: 为什么看不到某些功能？**
A: 您的角色可能没有相应权限，请联系管理员分配权限。

**Q: 如何获取管理员权限？**
A: 管理员权限需要由现有管理员分配，请联系系统管理员。

### 9.3 数据问题

**Q: 数据导入失败怎么办？**
A: 
1. 检查文件格式是否正确
2. 检查必填字段是否填写
3. 检查数据格式是否符合要求
4. 查看错误提示信息

**Q: 如何同步多台电脑的数据？**
A: 使用数据库配置功能，连接同一个远程数据库即可自动同步。

### 9.4 系统问题

**Q: 页面加载缓慢怎么办？**
A: 
1. 检查网络连接
2. 清除浏览器缓存
3. 关闭不必要的浏览器标签

**Q: 消息通知不显示？**
A: 
1. 检查浏览器通知权限
2. 检查系统通知设置
